Shell, Initialization Files, Variables and Expansions

This project covers the basics of the Bash shell environment, including how the shell initializes, how to manage variables (local and global), and the power of shell expansions and aliases.

## Learning Objectives
* Understanding shell initialization files (`/etc/profile`, `~/.bashrc`).
* Distinguishing between local and global variables.
* Mastering shell expansions and arithmetic.
* Creating and managing aliases.
